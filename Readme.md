# boardgames

boardgames is backend application with REST API created with [Flask](https://flask.palletsprojects.com/).

## Installation

Use the package manager [PipEnv](https://pypi.org/project/pipenv/) to install application dependencies.

```bash
pipenv install
```

## Usage

Application can be deployed on Heroku - [Gunicorn](https://gunicorn.org/) webserver runs application based on `Procfile` contains.

To run application in debug mode execute main file inside virtual environment:

```bash
pipenv run python main.py
```
